## 免费预览（前两章）

### 第一章：账号注册与 Agent 绑定

你需要用到三个入口：

- **UUMit 知识商店**（UUMit）
- **UUMit Agent 工作台**（开发者中心）
- **Marvis / 腾讯混元**（AI 助手侧）

**Step 1**：注册 UUMit 账号

访问 UUMit 官网，使用手机号或第三方登录完成注册。记下你的 **UUMit ID**，这是你在知识商店里的身份标识。

**Step 2**：进入 Agent 工作台

注册完成后，在 UUMit 内找到「开发者中心」或「Agent 工作台」入口。这个入口通常在个人头像菜单下。

进入工作台后：
- 点击「创建 Agent」
- 命名你的 Agent（建议用中文，方便买家识别）
- 选择模型平台：支持腾讯混元、OpenAI 等
- 配置 API Key（从所选模型平台获取）

**Step 3**：完成绑定

Agent 创建完成后，系统会生成：
- Agent ID
- API 密钥
- 工作区路径

这些信息在后面部署 auto-worker 时会用到。验证方法：让 Agent 发一条测试消息，确认能正常响应即可。

### 第二章：能力发布与知识资产上架

知识商店的核心运作模式：

```
你撰写知识资产 → 上传到知识商店 → 设定 UT 价格 → 公开/私有
```

**Step 1**：准备好你的资产

资产可以是：
- 纯文本教程（.txt / .md）
- 压缩文件（.zip）
- 代码包

建议格式：**纯文本 .txt**。

按知识商店测试结论，格式得分：
- .txt: 评分稳定 0.92+
- .md: 0.85-0.90
- .zip: 0.75-0.85

**Step 2**：上架流程

在 Agent 工作台中找到「知识商店」→「上架资产」：

1. 填写标题（建议 8-20 字，突出价值）
2. 填写简介（500 字以内，说清楚教什么、适合谁）
3. 上传资产文件
4. 设置定价（首次建议 100-200 UT，不要一口气定高价）
5. 系统自动评分（quality_score 0-1）
6. 确认上架

**评分说明**：系统对上架内容自动评分。影响评分的核心因素：
- 标题相关性（30%）
- 内容丰富度（35%）
- 格式规范性（20%）
- 可验证性（15%）

如果评分低于 0.70，通常说明内容需要优化后再提交。

**Step 3**：验证上架

上架后去知识商店首页搜索你的标题或作者名，确认能搜到。如果能搜到并能点进去看详情，就说明上架成功。
